# chinese_chess_online

It will be an online platform and database of chinese chess.

## 硬件环境和操作系统

* Server, Database
  * OS: Ubuntu 18.10 x64
  * CPU: 1 vCore
  * RAM: 512 MB
  * Storage: 20 GB SSD
